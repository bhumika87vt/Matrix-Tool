# Matrix_Tool
Matrix Tool is a lightweight and easy-to-use application designed to perform essential matrix operations for students, developers, and anyone working with linear algebra. The tool provides a clean interface and fast computations, helping users solve matrix problems without manual calculations.

Features:
1. Create matrices of any size
2. Perform basic operations: addition, subtraction, multiplication
3. Compute determinant, transpose, and inverse
4. Error-handling for invalid matrix operations
5. Fast and accurate results
6. Simple UI with clear input/output layout

Tech Stack
Frontend: React 18 + CSS Grid + Flexbox
Backend: Node.js/Express API (localhost:5000)
Styling: Custom CSS (no frameworks)
Deployment: Ready for Vercel/Netlify

How It Works:
1. User inputs the matrix dimensions.
2. Fills in the matrix values.
3. Selects the desired operation.
4. Tool processes the input and displays results instantly.
